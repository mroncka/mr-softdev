# Brand Identity — MR Softdev

## Core Identity

**Tagline:** Real software. Real problems. Built in public.

**Voice:** Direct, technical, occasionally witty. No fluff. Show the work.

**Audience:** Developers curious about AI-augmented workflows, indie SaaS builders, tech enthusiasts.

---

## Visual Direction

- **Aesthetic:** Dark background, clean typography, electric accent (cyan `#00E5FF` or electric blue `#4FC3F7`)
- **Feel:** Terminal meets creative studio — functional but with personality
- **Not:** Corporate, tutorial-channel, hustle-bro

## Tone References

- Shows real systems, not toy examples
- Mentions production deployments and actual failures
- Has opinions about tools
- Builds at hardware/cloud/AI intersection — uncommon combo
